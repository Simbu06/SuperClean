# SuperClean - React Application

A modern, responsive cleaning services website built with React. This project showcases a professional cleaning service business with features like service listings, team members, and contact information.

## Overview

SuperClean is a fully responsive React application that was converted from a static HTML/CSS/JS website. It features a clean and intuitive user interface designed to showcase cleaning services effectively.

## Features

- **Fully Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern React Architecture**: Built with React components for maintainability and reusability
- **Smooth Scrolling**: Enhanced user experience with smooth navigation between sections
- **Mobile-Friendly Navigation**: Hamburger menu for easy mobile navigation
- **Service Showcase**: Highlight your cleaning services with dedicated sections
- **Team Display**: Introduce your team members with profile cards
- **Contact Information**: Easy-to-find contact details and newsletter signup

## Technologies Used

- **React 18**: Modern JavaScript library for building user interfaces
- **Create React App**: Standard tooling for React projects
- **CSS3**: For styling and responsive design
- **Boxicons**: Icon library for UI elements
- **Font Awesome**: Additional icon support
- **Google Fonts (Poppins)**: Professional typography

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. **Clone the repository:**

```bash
git clone https://github.com/Simbu06/SuperClean.git
cd SuperClean
```

2. **Install dependencies:**

```bash
npm install
```

3. **Start the development server:**

```bash
npm start
```

The app will open in your browser at [http://localhost:3000](http://localhost:3000)

## Deployment

This React app can be deployed to various platforms:

### Netlify (Recommended)

1. Push your code to GitHub
2. Connect your repository to Netlify
3. Set build command to `npm run build`
4. Set publish directory to `build`
5. Deploy!

### Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts

### Traditional Hosting

1. Run `npm run build`
2. Upload the contents of the `build` folder to your web server
